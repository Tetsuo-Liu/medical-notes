# 呼吸器
- [ ] 気管 → 気管支 → 細気管支 → *終末細気管支* → 呼吸細気管支 → 肺胞
- [ ] 気道のうち線毛があるのは気管・気管支・細気管支・終末細気管支
- [ ] 気道のうち軟骨があるのは気管・気管支
- [ ] 気管・気管支の上皮は*多列円柱線毛上皮*、細気管支以降は*単層立方上皮*
- [ ] 分泌細胞は、*気管・気管支*では*杯細胞*、*細気管支・終末細気管支*では*Clab細胞*、*呼吸細気管支・肺胞*では*AT2*
- [ ] 気管分岐部は*T4/T5*の高さ、体表では*胸骨角*
- [ ] 酸素解離曲線で重要な数値二つ
	- [ ] *SaO2* *90*% = *PaO2* *60*mmHg
	- [ ] *SaO2* *75*% = *PaO2* *40*mmHg：*混合静脈血*の酸素濃度に相当
- [ ] *2,3-DPG*の増加で酸素解離曲線は*右方偏位*する
- [ ] 慢性的な低酸素状態への代償で酸素解離曲線は*右方*偏位する
- [ ] Stridorとは*吸気*性喘鳴で*連続*性ラ音に分類される。*代表疾患は急性咽頭蓋炎*、*クループ*、*気道異物*
- [ ] II型呼吸不全ではPaCO2は > *45* mmHg
- [ ] A-aDO2の簡易計算式は PAO2-PaO2 = *150 - PaCO2/0.8 - PaO2*、正常値は*5-15* Torr
- [ ] 口すぼめ呼吸は*COPD*の特徴的所見
- [ ] 肺胞換気量について
	- [ ] 定義：*1分間で肺胞に到達し、ガス交換に寄与する有効な空気量*
	- [ ] 計算式：*（一回換気量-死腔）× 呼吸数/min*
- [ ] COPDの代償期において、肺胞換気量は*変化しない*
- [ ] LAMA投与時の禁忌は、*閉塞隅角緑内障*、*前立腺肥大症*
- [ ] 肺サルコイドーシスに特徴的な検査所見：
	- [ ] 血清*ACE値の上昇*
	- [ ] *高Ca血症*（他のサルコイドーシスでも起こる）
	- [ ] *ツ反陰転化*（*ホジキンリンパ腫*などでも起こる）
