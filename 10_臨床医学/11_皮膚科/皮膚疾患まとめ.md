# 炎症性皮膚疾患
## 湿疹（皮膚炎）の病態と検査を述べよ
- 病態：表皮を中心とした炎症。
- 臨床経過：紅斑 → 丘疹・小水疱 → 湿潤・結痂 → 苔癬化・落屑 と多彩な皮疹が混在する。
- 疾患：
	- 接触皮膚炎
		- 病態：*Ⅳ*型アレルギー
			- MEMO：原因となるアレルゲンはほとんど*ハプテン*なので
		- 検査：原因検索には*パッチテスト*を行う。
	- アトピー性皮膚炎
		- 検査：*白色皮膚描記法*が陽性となる。
	- 他に*皮脂欠乏性湿疹*など。

## 蕁麻疹の病態と検査を述べよ
- 病態：
	- *真皮*における一過性の浮腫（*膨*疹）
	- *Ⅰ*型アレルギーが関与する
- 臨床像：瘙痒を伴う*膨*疹が出現し、通常24時間以内に消退する。
- 検査：*赤色皮膚描記法*が陽性となる。
	- 機序：物理的刺激によって*真*皮に存在する*マスト細胞*が過剰に反応するため。

## 炎症性角化症の代表疾患と特徴を述べよ
- 病態：
	- *角層の肥厚*を伴う炎症性疾患。
	- *[[皮膚疾患まとめ#皮膚科の理学検査|ケブネル現象]]* が陽性となる。
- 具体的な疾患：
	- *尋常性乾癬*
		- 機序：*表皮のターンオーバー亢進*
		- 皮疹：境界明瞭な紅色局面に銀白色の鱗屑が付着する。
		- 物理所見：*[[皮膚疾患まとめ#皮膚科の理学検査|アウスピッツ現象]]* が陽性。
	- *扁平苔癬*
		- 機序：*CTLが基底細胞破壊→顆粒層が不均一に肥厚*
		- 皮疹：*紫紅色*の扁平な丘疹。*Wickham線条*を認める。
			- *Wickham線条*：*網目状・レース状の白色線条*

## 脂漏性角化症について述べよ
- 分類：*表皮角化細胞*の増殖による*良*性腫瘍。
- 臨床像：*高齢者*の顔面や体幹に好発する、褐色～黒色の隆起性病変。*老人性疣贅*とも呼ばれる。
- 随伴徴候：短期間に*瘙痒*を伴って急に多発した場合、*内臓悪性腫瘍*（特に*胃がん*など）の合併を考慮する（*Leser-Trélat徴候*）。
（参考）[[Pasted image 20251004094827.png]]

## 結節性紅斑について述べよ
- 病態：*皮下脂肪組織*の炎症。
- 皮疹：*下腿伸側*に好発する*有痛性*の*皮下硬結*を伴う紅斑。
- 関連疾患：*[[ベーチェット病]]*、*[[サルコイドーシス]]*、*炎症性腸疾患*（IBD）
（参考）[[Pasted image 20251004094808.png]]

## 硬結性紅斑について述べよ
- *結核菌の感染*が原因。*下腿屈側*に好発。*脂肪組織*の炎症で、*血管炎*を伴う。

# 水疱症と膿疱症
## 天疱瘡の病態と種類、病理所見を述べよ
- 病態：自己抗体が*デスモゾーム*を攻撃し、*表皮内水疱*を形成する。
- 検査所見：*[[皮膚疾患まとめ#皮膚科の理学検査|ニコルスキー現象]]* が陽性。
- 検査：*[[皮膚疾患まとめ#皮膚科の理学検査|Tzanck試験]]* で*棘融解細胞*を認める。
- 具体的な疾患：
	- *尋常性*天疱瘡
		- 特徴：*弛緩性水疱*で、*口腔内びらん*を初発とすることが多い。
		- 病理：*基底層直上*での表皮解離 → *大きな*水疱
	- *落葉状*天疱瘡
		- 病理：*角層下*での表皮解離 → *小さな*水疱（*ほとんど水泡は見えず、落屑が主体。口腔粘膜は侵されない*）
（参考）[[Pasted image 20251004100634.png]]

## 水疱性類天疱瘡の病態と病理所見を述べよ
- 病態：自己抗体が*ヘミデスモゾーム*を攻撃し、*表皮下水疱*を形成する。
- 臨床像：*高齢者*に好発。*緊満した*水疱で、*強い瘙痒*を伴う。
- 検査所見：*ニコルスキー現象陰性*
（参考）[[Pasted image 20251004100818.png]]

## 掌蹠膿疱症の病態と関連因子を述べよ
- 臨床像：手掌と足蹠に*無菌性*膿疱が多発する。
- 関連因子：
	- *慢性扁桃炎*
		- 機序：*溶連菌に対するT細胞の交差反応性による自己免疫*
	- *歯科金属アレルギー*
		- 機序：*金属は特にエクリン汗腺から排泄されやすいが、排泄時に濃度が上がり、エクリン汗腺が攻撃されるため。IV型アレルギー*
	- *喫煙*
## 壊疽性膿皮症について述べよ
- 臨床像：急速に拡大する有痛性の潰瘍
- 関連疾患：*炎症性腸疾患*（*IBD*）、関節リウマチ、骨髄異形成症候群（MDS）などに合併することが多い。

# 母斑と母斑症
## 母斑の分類と特徴を述べよ
- 母斑：皮膚の色調や形に異常を呈する限局性の病変。
	- *メラノサイト系母斑*
		- *蒙古斑*：臀部にみられる青色斑。自然消退する。
		- *太田母斑*：*顔面の三叉神経第1, 2枝領域*にみられる青褐色斑。自然消退せず、*レーザー治療*の対象。
		- *色素性母斑*：いわゆるほくろ。基本的には経過観察。
	- *類器官母斑*
		- *脂腺母斑*：頭部や顔面に好発する黄褐色の局面。将来的に*基底細胞癌*を生じるリスクがあるため、切除が推奨される。
	- *血管系母斑（血管腫）*
		- *乳児血管腫*（*イチゴ状血管腫*）：*毛細血管内皮*細胞の増殖による赤色腫瘤。生後増大し、*自然消退*する。
		- *単純性血管腫*：*真皮毛細血管の拡張*による赤色斑。
			- *サーモンパッチ*：*前額部正中*にできる。自然消失*する*。
			- *ウンナ母斑*：*項部*にできる。自然消失傾向。
			- *Sturge-Weber症候群*：*顔面片側*（特に*V1領域*）の*ポートワイン母斑*

## 母斑症の分類と特徴を述べよ
- 母斑症（*神経皮膚*症候群）：母斑が皮膚だけでなく、神経系など全身の器官にみられる疾患群。
	- *神経線維腫症1型*（von Recklinghausen病）
		- 原因：*Ras*経路の抑制因子である*NF1*遺伝子の変異。*常染色体顕性*遺伝。
		- 皮膚所見：*カフェオレ斑*、*神経線維腫*
		- 全身症状：*虹彩結節*（Lisch結節）、*脊椎側弯*など。
	- *結節性硬化症*（Pringle病）
		- 原因：*mTOR*経路の抑制因子である*TSC*遺伝子の変異。*常染色体顕性*遺伝。
		- 皮膚所見：*葉状白斑*、*顔面の血管線維腫*
		- 全身症状：
			- *West症候群*（*乳幼児に発症する点頭てんかん*）
			- *精神発達遅滞*
			- *脳内石灰化*
			- *心横紋筋腫*、*腎血管筋脂肪腫*、*リンパ脈管筋腫症*など
	- *Sturge-Weber症候群*
		- 原因：GNAQ遺伝子の体細胞モザイク
		- 皮膚所見：*顔面片側*（特に*V1領域*）の*ポートワイン母斑*
		- 全身症状：
			- *同側の緑内障*
			- *脳軟膜血管腫によるてんかん*
			- *精神発達遅滞*
# 皮膚科の理学検査
## 皮膚科に特有の物理的徴候を挙げ、関連疾患を述べよ
- ニコルスキー現象：*健常皮膚を擦ると水疱形成*
	- 陽性になる疾患：*天疱瘡*、*SSSS*、*TEN*
- ケブネル現象：*健常皮膚を刺激すると同種の皮疹が出現*
	- 陽性になる疾患：*尋常性乾癬*、*扁平苔癬*
- アウスピッツ現象：鱗屑を剥離すると点状出血
	- 陽性になる疾患：*尋常性乾癬*
- 皮膚描記法：
	- 赤色描記法 → *蕁麻疹*
	- 白色描記法 → *アトピー性皮膚炎*

## 皮膚科で用いる簡便な検査法を挙げ、目的を述べよ
- Tzanck試験：水疱底の細胞を顕鏡
	- 疾患と所見：
		- 天疱瘡では*棘融解細胞*を認める
		- HSV感染症（口唇ヘルペス、性器ヘルペス、Kaposi水痘様発疹症）やVZV感染症で*多型巨細胞*
	- 参考）[[Pasted image 20251004151816.png]]
- 硝子圧法：皮疹をガラスで圧迫
	- *褪色すれば血管拡張による紅斑*
	- *褪色しなければ血管外漏出による紫斑*
- 苛性カリ顕鏡法：鱗屑をKOHで溶解して顕鏡 → *真菌*（*白癬菌*、*カンジダ*）*の菌糸や胞子*を確認する
- 最小紅斑量試験：紅斑を生じさせる最小の*光線量*を見る
	- 疾患：*光線過敏症*
- ダーモスコピー：色素性病変を拡大視 → *色素性母斑*と*悪性黒色腫*の鑑別
	（参考）[[Pasted image 20251004152139.png]]
