## カドミウム中毒により骨粗鬆症が生じる機序は？
*近位尿細管*障害 → *Vit.D活性化*障害・*CaとPの再吸収*障害 → 骨粗鬆症
## パラコートの初期治療における禁忌は？★
*酸素投与*。*ラジカル発生*が原因のため
## 中毒の死因と死斑の色調の関係★

| 中毒の原因     | 死斑の色調 |
| --------- | ----- |
| 通常        | *暗紫色* |
| 一酸化炭素中毒   | *鮮紅色* |
| 青酸中毒      | *鮮紅色* |
| 硫化水素      | *緑褐色* |
| 塩素酸塩、亜硝酸塩 | *暗褐色* |
## オピオイドの離脱で見られる症状は？★
*縮瞳*、*発汗*、*頻脈*、*下痢*、*流涎*

## 有機リン中毒の症状は？
- 機序：*AchEの阻害*
- 症状：*ムスカリン*様作用が主。*縮瞳、流涎、流涙、発汗、下痢*など
	- 注意点：頻脈も徐脈も生じうる。
